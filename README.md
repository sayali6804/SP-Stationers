# SP-Stationers

A full-stack e-commerce web application for managing an online stationery store. The platform enables customers to browse products, place orders, manage their shopping cart, and securely authenticate, while providing administrators with inventory and order management capabilities.

---

## Overview

SP-Stationers is designed to streamline the online stationery shopping experience by providing an intuitive user interface, secure authentication, efficient product management, and seamless order processing. The application follows a modular architecture with RESTful APIs, ensuring scalability, maintainability, and a responsive user experience.
